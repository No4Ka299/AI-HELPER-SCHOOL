# AI HELPER SCHOOL

AI HELPER SCHOOL - это веб-сайт, который создает академические работы по ГОСТу: курсовые работы (25+ страниц), сочинения (300+ строк) и доклады (2+ страницы).

## Особенности

- Создание курсовых работ объемом 25+ страниц по ГОСТу
- Создание сочинений объемом 300+ строк
- Создание докладов объемом 2+ страниц
- Использование информации из научных источников (в том числе с сайта cyberleninka.ru)
- Современный интерфейс с возможностью настройки параметров работы
- Возможность скачивания готовой работы в формате Word
- Соответствие требованиям ГОСТ для академических работ
- Бесплатное использование всех функций

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)

## Установка и запуск

1. Скопируйте файлы проекта
2. Откройте `index.html` в веб-браузере

## Использование

1. Выберите тип работы (курсовая, сочинение или доклад)
2. Введите тему работы
3. Укажите предмет
4. При необходимости укажите количество страниц
5. Добавьте дополнительные требования (по желанию)
6. Нажмите "Создать работу"
7. После генерации работы вы можете скачать её в формате Word

## Структура проекта

- `index.html` - основная HTML-страница с интерфейсом
- `styles.css` - стили для веб-сайта
- `script.js` - JavaScript-логика генерации работ по ГОСТу
- `README.md` - документация проекта

## Функциональность

- Генерация структурированных академических работ
- Автоматическое форматирование по ГОСТу
- Добавление титульных страниц, содержания, списка источников
- Генерация контента на основе научных источников
- Возможность настройки объема работ
- Экспорт в формат Word для дальнейшего редактирования
