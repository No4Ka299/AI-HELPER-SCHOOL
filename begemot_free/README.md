# FreeBegemot - Бесплатная альтернатива Begemot.ai

## Описание
FreeBegemot - это бесплатная альтернатива популярному сервису Begemot.ai, которая позволяет генерировать академические работы с помощью нейросетей. Проект создан с целью предоставления студентам и школьникам бесплатного доступа к инструментам для создания проектов, рефератов, курсовых и других видов работ.

## Особенности
- Генерация действительно уникального текста с использованием расширенных шаблонов
- Создание структурированных работ (введение, основная часть, заключение, список литературы)
- Возможность экспорта в различные форматы
- Создание презентаций в формате .pptx
- Иллюстрации к разделам работ
- Современный и интуитивно понятный интерфейс

## Структура проекта
```
begemot_free/
```

## Установка и запуск
1. Клонируйте репозиторий
2. Установите локальный веб-сервер (например, Apache с поддержкой PHP)
3. Поместите файлы проекта в корневую директорию сервера
4. Откройте в браузере главную страницу

## Использование
1. Введите тему работы в поле ввода
2. Нажмите кнопку "Создать"
3. Дождитесь генерации результата
4. Просмотрите и при необходимости экспортируйте результат
