# Страницы сайта FreeBegemot

## Главные страницы

### index.html
- Главная страница сайта
- Позволяет создавать индивидуальные проекты
- Содержит информацию о возможностях и ценах
- Имеет форму для ввода темы проекта

### referat.html
- Страница для генерации рефератов
- Позволяет создавать работы объемом 10-15 страниц
- Содержит структуру реферата с введением, основной частью и заключением

### course-work.html
- Страница для генерации курсовых работ
- Позволяет создавать работы объемом 20-25 страниц
- Включает теоретическую и практическую части

### result.html
- Страница с результатами генерации
- Отображает созданный текст в структурированном виде
- Позволяет экспортировать результат в различные форматы

## Вспомогательные файлы

### api/generate.php
- API-эндпоинт для генерации текста
- Принимает тему работы и возвращает сгенерированный текст
- Имитирует работу нейросети

### css/style.css
- Основные стили сайта
- Ответственны за визуальное оформление

### js/main.js
- Основной JavaScript-код
- Обрабатывает события, отправляет запросы к API
- Управляет интерактивными элементами

### images/
- Директория с изображениями и иконками
- Содержит логотипы, иконки и иллюстрации

## Особенности реализации

- Все страницы имеют единый дизайн и навигацию
- При генерации текста используется имитация нейросети
- Результаты сохраняются в localStorage перед переходом на страницу результатов
- Поддерживается адаптивный дизайн для разных устройств
- Все функции бесплатны для пользователя
